## Как запускать бота

1. Запустить Docker Compose
   sudo docker-compose -f database/docker-compose.yml up -d
2. Запустить бота
   python3 /bot/bot.py&

## Как деплоить сайт

1.	Получить доступ к VPS через SSH под пользователем с правами root.
	2.	Установить и настроить веб-сервер (например, Apache или Nginx).
	3.	Создать пользователя и загрузить файлы сайта на сервер (через SFTP или SCP).
	4.	Настроить доменное имя, чтобы оно указывало на IP вашего VPS.
	5.	Если сайт использует базу данных, создать базу и импортировать дамп.
	6.	Настроить SSL-сертификат для защищённого соединения (например, с помощью certbot).
	7.	Запустить сайт и проверить, что всё работает корректно.
